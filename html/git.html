Git

  • git init — инициализирует новый Git-репозиторий в текущей директории.
  • git status — показывает текущее состояние репозитория и изменённые файлы.
  • git add — добавляет файлы в индекс (staging area) перед коммитом.
  • git commit — сохраняет изменения в репозитории с комментарием.
  • git push — отправляет локальные коммиты в удалённый репозиторий.
